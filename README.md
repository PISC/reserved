# piscpr/reserved

Vendor namespace reservation for the `piscpr/*` Composer package family.

This repository exists solely to claim the `piscpr` vendor name on Packagist so it
cannot be squatted. The real `piscpr/*` packages live in private repositories
under the [PISC GitHub organization](https://github.com/PISC) and are consumed
via path or private Composer repositories — they are not published to Packagist.

Do not install this package. It is a metapackage with no code and no dependencies.
